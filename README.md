# Проект для добавления автоматизации

Этот проект создан для более простого добавления автоматизации по отправке уведомлений с камеры FriGate в мессенджер MAX через интеграцию [MaxNotify](https://github.com/kai-zer-ru/max-notify-ha)

Добавить в HomeAssistant: [![Import blueprint](https://my.home-assistant.io/badges/blueprint_import.svg)](https://my.home-assistant.io/redirect/blueprint_import/?blueprint_url=https://raw.githubusercontent.com/kai-zer-ru/freegate_camera_notify_blueprint/refs/heads/main/blueprint.yaml)
